# 炼金合成逻辑总结

> 2026-07-04 基于代码、文档与 14 组游戏实测数据整理。本文档作为后续开发的唯一真理源。

## 一、物等范围

```
L_base = min(所有材料物等)

L_base ≥ 8:  降级范围 = 7 级 → 下界 = max(1, L_base - 7)
L_base < 8:  降级范围 = 3 级 → 下界 = max(1, L_base - 3)

上界 = L_base + 4 + 百科等级
```

| 百科等级 B | 上限加成 | 示例 (L_base=5) |
|---|---|---|
| 0 | +0 | 2 ~ 9 |
| 1 | +1 | 2 ~ 10 |
| 2 | +2 | 2 ~ 11 |
| 3 | +3 | 2 ~ 12 |
| 4 | +4 | 2 ~ 13 |

**百科只扩展上界，不影响下界。**

## 二、候选筛选（三步过滤）

### Step 1 — 类型 + 主属性 + 等级

```
item.material === 主材材质
item.level ∈ [下界, 上界]
(item.req_level > 0) || (item.category === "矿石")
```

### Step 2 — 副材钥匙规则

对每个非主材的输入材质，依次检查：

```
if 该材质 ∈ 任何剩余候选的子属性:
    → 该材质是"钥匙" → 只保留需要该子属性的候选
else:
    → 该材质是"杂物" → 不过滤
```

**关键**：单属性物品（无子属性要求）只在所有副材都是杂物时出现。

### Step 3 — 子属性覆盖

```
产物的所有子属性 ⊆ 输入材料集合
```

## 四、概率分布（未完成）

当前 `getDeltaProb`:

| delta (L - L_base) | 概率 |
|---|---|
| +4 | 10% |
| +3 | 25% |
| +2 | 30% |
| +1 | 15% |
| 0 | 15% |
| -1 ~ -N | 5% / N (N=降级范围) |
| **+5 ~ +8** | **未定义 (百科延伸)** |

存在问题：
- d>4 (百科延伸等级) 概率为 0
- 有百科时实测产出常落在 d=+5/+6，模型给 0%
